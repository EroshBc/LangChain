change name to .env
1. Open Finder and go to your folder.
2. If you don't see .env.txt, press Command + Shift + . (Period) to reveal hidden files.
3. Click the file name, press Return (Enter) on your keyboard to edit the name.
4. Delete the .txt from the end so it just says .env and press Enter.
5. Mac will pop up a warning asking if you are sure:
6. Click "Use ." to confirm.
